Must-Know Limits

Gotchas & Rules

Write these down. Break these rules and your automations fail silently.

1. Follow-to-DM trigger is in Beta
2. First Welcome Message must have a button (24hr window)
3. First private reply = one content block only
4. Cannot trigger on your own comments
5. Story replies only work on stories < 24 hours old
6. Rate limits on viral posts — pause if needed
